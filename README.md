# SP-2025

This repository contains C++ programs developed as part of the learning journey.

## Programs

- `even_odd_sums_ratio.cpp`: Calculates the ratio of sums of even and odd numbers.
- `matrix_sum_diff.cpp`: Computes sum and difference of matrices.
- `x_shape_counter.cpp`: Counts X shapes in a matrix.

## How to Compile and Run

Use a C++ compiler like g++.

Example:
```
g++ even_odd_sums_ratio.cpp -o even_odd_sums_ratio.exe
./even_odd_sums_ratio.exe
```
